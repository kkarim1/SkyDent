# SkyDent Website

هذه نسخة عامة آمنة للزوار، ولا تحتوي على كلمة مرور أو لوحة إدارة داخل ملفات الموقع.

## الملفات التي ترفعها

ارفع محتويات مجلد `skydent-site` بالكامل:

- `index.html`
- `style.css`
- `app.js`
- `assets/`

## بخصوص لوحة الإدارة

لا تضع كلمة مرور الإدارة داخل HTML أو JavaScript في موقع ثابت، لأن أي زائر يستطيع فتح الكود ومعرفتها.

لعمل لوحة إدارة حقيقية لتعديل معلومات الاتصال ونشر الصور، استخدم باك إند آمن مثل:

- Firebase Authentication + Firestore + Storage
- Supabase Auth + Database + Storage
- Laravel / Node.js API مع قاعدة بيانات
- WordPress بلوحة تحكم مخصصة

بعد ذلك تكون لوحة الإدارة على رابط منفصل ومحمي، والموقع العام يسحب البيانات منها فقط.
